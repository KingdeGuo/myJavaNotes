# 一些都是对象

- 五个存储数据的地方
  - 寄存器
    - 速度最快
    - 不能直接控制
  - 堆栈
    - 通常位于RAM中（随机访问存储器）
    - Java系统必须知道存储在堆栈内部所有项的确切生命周期，以便上下移动指针。
  - 堆
    - 通常位于内存池（也位于RAM中）。
    - 编译器不需要知道数据在堆里存活多长时间。
  - 常量存储
    - 通常直接放在程序代码内部
  - 非RAM存储
    - 数据完全存活于程序之外。
    - 两个基本例子是流对象和持久化对象
      - 流对象：转化成字节流发送到另外一台机器上
      - 持久化对象：把对象存在磁盘上。
